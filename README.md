# 项目简介

## 核心功能
- 货物管理
- 入库记录
- 出库记录
- 个人中心
- 登录注册

## 适用场景
- 仓库管理员
- 货主
- 采购人员

## 技术特点
- 使用Django框架构建
- 数据库采用MySQL
- 前端技术包括HTML、CSS、JavaScript
